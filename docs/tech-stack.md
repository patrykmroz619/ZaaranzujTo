# Tech Stack

## Frontend

- **Framework:** Next.js 16
- **State Management:** Zustand
- **Styling:** Tailwind CSS v4
- **Auth:** Clerk (Client SDK)

## Backend

- **Framework:** Nest.js
- **Database:** MongoDB
- **ODM:** Mongoose
- **API Style:** REST API
- **Auth:** Clerk SDK (Server-side JWT verification)
- **AI/LLM:** OpenRouter API (LLM & Vision models)
- **Request Validation:** Zod
