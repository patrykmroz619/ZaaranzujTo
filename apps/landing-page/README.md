# landing-page

Public marketing landing page for ZaaranżujTo, built with Astro + React 19 + Tailwind CSS v4.

## Commands

Run from the monorepo root:

```sh
bun run dev:landing            # dev server on http://localhost:3001
bun run --filter=landing-page build
bun run --filter=landing-page preview
bun run --filter=landing-page check-types
```
